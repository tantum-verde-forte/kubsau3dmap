init()

Главная функция программы. В данной функции происходит создание сцены и работа с её объектами.
Properties:
Name Type Description
scene THREE.Scene объект на котоором происходит рендер графики
camera THREE.PerspectiveCamera объект с помощью которого формируется ракурс картинки, которая отрисовывается на scene
controls THREE.OrbitControls инструмент с помощью которого происходит управление camera
light THREE.AmbientLight освещение добовляемове на сцену
renderer THREE.WebGLRenderer инструмент с помощью которого отрисовывается изображение на экране
cube THREE.Mesh объект создающий задний фон
sprite THREE.Sprite объект, который графически указывает на здание
loader THREE.GLTFLoader загружает объект на сцену
